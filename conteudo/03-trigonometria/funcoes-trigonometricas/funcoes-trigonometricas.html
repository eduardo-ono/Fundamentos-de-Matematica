2. Funções Trigonométricas¶

2.1. Definição¶

Funes trigonomtricas so aplicaes de $\mathbb{R}$ sobre o ciclo trigonomtrico $\lambda$, onde podemos associar a cada nmero real $x$ um nico ponto $P$ de $\lambda$ do seguinte modo:

  1. Se $\,x = 0\,$, ento $P$ coincide com $A$;

  2. Se $\,x > 0\,$, ento realizamos a partir de $A$ um percurso de comprimento $x$, no sentido anti-horrio, e marcamos $P$ como ponto final do percurso;

  3. Se $\,x < 0\,$, ento realizamos a partir de $A$ um percurso de comprimento $|x|$, no sentido horrio. O ponto final do percurso $P$.
Fonte: Iezzi

O ponto $P$ a imagem dos elementos do conjunto

$$ \{x \in \mathbb{R}\ | \ x = x_0 + 2k\pi, \ k \in \mathbb{Z}\} $$
Fonte: Iezzi

3. Função Seno: $\ y = \text{sen }x\ $¶

3.1. Definição¶

Denominamos funo seno a funo $\,f: \mathbb{R} \longrightarrow \mathbb{R}\,$ que associa a cada real $x$ o real $\,OP_1 = \text{sen }x\,$, isto :

$$ f(x) = \text{sen }x $$

3.2. Gráfico¶

import numpy as np
import math

x_min = -2 * math.pi
x_max = 2 * math.pi
x_step = (x_max - x_min) / 1000
y_min = -1
y_max = 1
plt.xlabel("$x$ [rad]", fontsize=14)
plt.ylabel("$sin(x)$", fontsize=14)
x_values = np.arange(x_min, x_max + x_step, x_step)
y_values = np.sin(x_values)

plt.grid()
plt.axhline(y=0.0, color='grey', linestyle='-', linewidth=1)
plt.axvline(x=0.0, color='grey', linestyle='-', linewidth=1)
plt.xlim(x_min, x_max)
plt.ylim(y_min, y_max)
radian_multiples = np.arange(x_min / math.pi, x_max / math.pi + 0.5, 0.5) # [-2, -3/2, -1, -1/2, 0, 1/2, 1, 3/2]
radians = [n * math.pi for n in radian_multiples]
radian_labels = []
for n in radian_multiples:
    label = '0' if n == 0 else rf'{int(n)}$\pi$' if n == int(n) else rf'{n}$\pi$'
    radian_labels.append(label)

plt.xticks(radians, radian_labels)
plt.plot(x_values, y_values)
plt.show()

4. Função Cosseno: $\ y = \cos x\ $¶

4.1. Definição¶

Denominamos funo cosseno a funo $\,f: \mathbb{R} \longrightarrow \mathbb{R}\,$ que associa a cada real $x$ o real $\,OP_2 = \cos x\,$, isto :

$$ f(x) = \cos x $$

5. Função Tangente: $\ y = \text{tg }x\ $¶

Denominamos funo tangente a funo $\,f: D \longrightarrow \mathbb{R}\,$, que associa a cada real $x$, $\,x \neq \cfrac{\pi}{2} + k\pi\,$, o real $\,AT = \text{tg }x\,$, isto :

$$ f(x) = \text{tg }x $$

5.1. Propriedades¶

  1. O domnio da funo tangente $\,D = \left\{ x \in \mathbb{R} \ | \ x \neq \cfrac{\pi}{2} + k\pi, \ k \in \mathbb{Z} \right\}\,$.

  2. A imagem da funo tangente $\mathbb{R}$, isto , para todo $y$ real existe um $x$ real tal que $\,\text{tg }x = y\,$. De fa

  3. A funo tangente peridica e seu perodo $\pi$.

$$ \text{tg }x = \text{tg }(x + k\pi) $$
